II

Vector Spaces and Subspaces

The algebraic foundation of linear algebra

1. What is a Vector Space?

Definition A Vector Space V over a field F (usually ℝ) is a set of objects (called vectors) with two operations — vector addition and scalar multiplication — that satisfy 10 axioms.

The 10 Axioms (must know!)

For all u, v, w ∈ V and scalars c, d ∈ F:

#AxiomWhat it means
A1u + v ∈ VClosed under addition
A2u + v = v + uCommutative
A3(u+v)+w = u+(v+w)Associative
A4∃ 0 : v+0=vZero vector exists
A5∃ -v : v+(-v)=0Additive inverse
S1cv ∈ VClosed under scalar mult.
S2c(u+v)=cu+cvDistributive over vector sum
S3(c+d)v=cv+dvDistributive over scalar sum
S4(cd)v=c(dv)Associative scalar mult.
S51·v = vIdentity scalar
Common Examples
  • ℝⁿ (all n-tuples of real numbers) — the most important
  • Mₘₓₙ — set of all m×n matrices
  • Pₙ — set of all polynomials of degree ≤ n
  • {0} — the trivial vector space

2. Subspaces

Definition A non-empty subset W ⊆ V is a subspace if it is itself a vector space under the same operations.
⚡ Subspace Test (3 conditions only!) W is a subspace of V if and only if:
  1. Non-empty: W ≠ ∅ (equivalently, 0 ∈ W)
  2. Closed under addition: u, v ∈ W ⟹ u + v ∈ W
  3. Closed under scalar multiplication: u ∈ W, c ∈ ℝ ⟹ cu ∈ W
Example — Is W = {(x,y) : x + y = 0} a subspace of ℝ²?
  1. (0,0): 0+0=0 ✓ → non-empty
  2. Take (a,-a) and (b,-b): sum = (a+b, -(a+b)) → still satisfies x+y=0 ✓
  3. c·(a,-a) = (ca, -ca) → still satisfies x+y=0 ✓
∴ W is a subspace.

3. Linear Independence

Definition Vectors v₁, v₂, …, vₖ are linearly independent if the only solution to c₁v₁ + c₂v₂ + … + cₖvₖ = 0 is c₁ = c₂ = … = cₖ = 0. Otherwise they are linearly dependent.
💡 Quick Check Method Put the vectors as rows (or columns) in a matrix. Row-reduce. If you get a zero row, the vectors are dependent; if no zero row, they are independent.
Example Are v₁=(1,2,0), v₂=(0,1,3), v₃=(1,3,3) linearly independent?
Matrix: [1 2 0] R3 → R3-R1 [1 2 0] [0 1 3] ──────────→ [0 1 3] [1 3 3] [0 1 3] R3 → R3-R2 → [0 0 0] ← zero row!
∴ Linearly dependent (v₃ = v₁ + v₂).

4. Span

Definition The span of vectors {v₁,…,vₖ} is the set of ALL linear combinations:
span{v₁,…,vₖ} = {c₁v₁ + c₂v₂ + … + cₖvₖ : cᵢ ∈ ℝ}
Span is always a subspace.

5. Basis and Dimension

Definition A basis of V is a set of vectors that is:
  1. Linearly independent
  2. Spans V
Dimension The dimension of V = number of vectors in any basis of V. Written dim(V).
SpaceStandard BasisDimension
ℝⁿe₁=(1,0,…,0), e₂=(0,1,…,0), … eₙn

7. Rank and Nullity

Definitions
  • Rank(A) = dim(Column Space) = dim(Row Space) = number of pivots
  • Nullity(A) = dim(Null Space) = n − rank(A)
🔑 Rank-Nullity Theorem (VERY IMPORTANT) For any m×n matrix A:
rank(A) + nullity(A) = n
(where n = number of columns)
Example A is 4×6 matrix with rank 3. Find nullity.
nullity = 6 − 3 = 3

8. Wronskian

Definition For functions f₁, f₂, …, fₙ, the Wronskian is the determinant:
W(f₁,…,fₙ) = |f₁ f₂ … fₙ | |f₁' f₂' … fₙ' | | ⋮ | |f₁⁽ⁿ⁻¹⁾ … fₙ⁽ⁿ⁻¹⁾|
If W ≠ 0 at some point → functions are linearly independent.
Example — Are sin x and cos x linearly independent?
W = |sin x cos x | |cos x -sin x | = sin x(-sin x) − cos x(cos x) = -sin²x − cos²x = -1 ≠ 0 ✓ Linearly INDEPENDENT

9. Application: Interpolation

Given n+1 data points, we find a polynomial of degree ≤ n passing through them. We write:

p(x) = a₀ + a₁x + a₂x² + … + aₙxⁿ

Substituting each point gives a system Ax = b. Solving this system (using row reduction or matrix inverse) gives the interpolating polynomial. The column space / basis ideas underpin why a unique polynomial exists.

III

Linear Transformations

Functions that preserve vector space structure

1. Definition

Definition A function T: V → W (from vector space V to W) is a linear transformation if for all u, v ∈ V and scalar c:
1. T(u + v) = T(u) + T(v) [Additivity] 2. T(cu) = cT(u) [Homogeneity]
Combined: T(cu + dv) = cT(u) + dT(v)
⚡ Quick Check Always verify: T(0) = 0. If T(0) ≠ 0, it's NOT linear!
Examples of Linear Transformations
NameDefinitionLinear?
Zero mapT(v) = 0✓ Yes
IdentityT(v) = v✓ Yes
ScalingT(x,y) = (2x, 3y)✓ Yes
Rotation by θT(x,y) = (x cosθ - y sinθ, x sinθ + y cosθ)✓ Yes
TranslationT(v) = v + b (b ≠ 0)✗ No (T(0)≠0)
T(x)=x²—✗ No

2. Kernel and Range

Kernel (Null Space) of T
ker(T) = {v ∈ V : T(v) = 0}
ker(T) is always a subspace of V.
Range (Image) of T
range(T) = {T(v) : v ∈ V} = {w ∈ W : w = T(v) for some v}
range(T) is always a subspace of W.
🔑 Dimension Theorem (Rank-Nullity for T)
dim(ker T) + dim(range T) = dim(V)
dim(ker T) = nullity, dim(range T) = rank

3. One-to-One and Onto

PropertyConditionEquivalent to
One-to-One (injective)T(u)=T(v) ⟹ u=vker(T) = {0}
Onto (surjective)range(T) = Wdim(range T) = dim W
Isomorphism (bijective)Both 1-1 and ontoV ≅ W (same structure)

4. Invertible Linear Transformations

Definition T: V→W is invertible if there exists T⁻¹: W→V such that T⁻¹∘T = Iᵥ and T∘T⁻¹ = Iᵥᵥ.
Theorem T is invertible ⟺ T is one-to-one AND onto ⟺ T is an isomorphism.

5. Matrix of a Linear Transformation

Every linear transformation T: ℝⁿ→ℝᵐ corresponds to a unique m×n matrix A such that:

T(x) = Ax for all x ∈ ℝⁿ
How to find the matrix A The j-th column of A is T(eⱼ), where eⱼ is the j-th standard basis vector.
Example T: ℝ²→ℝ² defined by T(x,y) = (x+2y, 3x−y). Find matrix A.
T(e₁) = T(1,0) = (1, 3) → column 1: [1, 3]ᵀ T(e₂) = T(0,1) = (2, -1) → column 2: [2, -1]ᵀ So A = [1 2] [3 -1] Check: A[x,y]ᵀ = [x+2y, 3x-y]ᵀ ✓

6. Change of Basis

If B = {b₁,…,bₙ} and C = {c₁,…,cₙ} are two bases of V, the transition matrix P_{B→C} converts coordinates from B to C.

[v]_C = P_{B→C} · [v]_B To find P_{B→C}: Write each basis vector of B in terms of C. Put those coordinate columns into a matrix.

7. Similarity

Definition Two matrices A and B are similar if there exists an invertible matrix P such that:
B = P⁻¹AP

Similar matrices represent the same linear transformation in different bases. They have the same eigenvalues, determinant, trace, and rank.
